# Talaqqi

## Overview
Talaqqi is an innovative online platform designed to make Qur’an learning accessible, authentic, and interactive for students everywhere. By combining the traditional teacher-student method of guided recitation with modern technology, the platform allows learners to receive live instruction while also benefiting from AI-powered feedback on their recitation.

Talaqqi bridges tradition and technology to create an intuitive, interactive, and effective Qur’an learning environment for the modern world, suitable for anyone seeking a personalized, reliable, and spiritually enriching learning experience.

## Features

- **User-Friendly Accounts & Security:** Easy account creation, secure login, and profile management.
- **Role-Based Access:** Distinct roles and dedicated dashboards for Students, Teachers, and Administrators.
- **Verified Teachers:** Ensuring a trusted and authentic learning environment.
- **Live Interactive Classes:** Real-time video and audio streaming for live sessions.
- **Synchronized Digital Mushaf:** Real-time synchronized viewing of the Qur’an for both students and teachers with dynamic highlighting.
- **Annotation & Notes:** Teachers can add, edit, and share annotations on verses to enhance student understanding.
- **AI-Powered Recitation Feedback:** Students can record their recitations and receive AI-driven Tajweed analysis and instant suggestions for improvement.
- **Progress Tracking & Recommendations:** Monitor mastery of recitation rules and receive personalized practice recommendations based on AI insights.

## Project Structure

The project is structured using plain HTML, CSS, and JavaScript.

### Core Pages
- `homePage.html`: Landing page explaining the platform's mission and features.
- `authentication.html` / `regForm.html`: User login and registration flow.
- `profile.html`: User profile viewing and management.

### Dashboards
- `stdDashboard.html`: Dashboard tailored for students.
- `tchDashboard.html`: Dashboard tailored for teachers.
- `admDashboard.html`: Dashboard for platform administrators.

### Learning & Interactive Interfaces
- `liveClass.html`: Interface for live, real-time educational classes.
- `recitation.html`: Qur'an recitation viewing, practicing, and recording.
- `aiFeedback.html`: Displays AI Tajweed analysis and actionable feedback.

### Management & Scheduling
- `schedule.html` / `scheduleClass.html`: Interface for scheduling and managing classes.
- `manageUsers.html` / `verifyTeachers.html`: Admin interfaces for managing users and verifying teacher credentials.
- `systemReports.html`: Administrator reporting dashboard.
- `studentProgress.html` / `recommendations.html`: Interface for tracking student metrics and viewing AI-driven recommendations.
- `editAnnotations.html`: Interface for teachers to manage and update verse annotations.

## Technologies Used
- **HTML5:** Semantic structuring of web pages.
- **CSS3:** Custom styling for a responsive and intuitive user interface.
- **JavaScript (Vanilla):** Client-side logic and basic interactivity.

## Getting Started

Since this is a vanilla frontend project, no complex build tools, package managers, or server setups are required to view the UI.

1. Clone or download the repository to your local machine.
2. Simply open `homePage.html` in your preferred web browser to begin exploring the platform.
